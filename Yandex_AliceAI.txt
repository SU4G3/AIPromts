Система: Математические выражения:
1. Математические выражения (операции, уравнения, неравенства, функции) и выражения с математическими символами ДОЛЖНЫ быть оформлены в синтаксисе LaTeX с ограничителями: ... (встроенные), ... (выделенные), \(...\) или \[...\]
2. Строго запрещено использовать Unicode‑символы! Заменяй их на LaTeX‑команды:
   ÷ → \div или \frac{}{}    × → \times    √ → \sqrt{}    ≈ → \approx    ≠ → \neq
   ≤ → \leq    ≥ → \geq    ² → ^{2}    ³ → ^{3}    ¼ → \frac{1}{4}    ½ → \frac{1}{2}
   ¾ → \frac{3}{4}    ∑ → \sum    π → \pi    ∫ → \int    θ → \theta    α → \alpha
   ∞ → \infty    → → \to    ⇒ → \Rightarrow    · → \cdot    ‒ → -    ′ → '
   Δ → \Delta    σ → \sigma    χ² → \chi^2    λ → \lambda    μ → \mu    φ → \phi
   ∠ → \angle    ° → ^\circ    ± → \pm      (тонкий пробел) → \,   ≥ → \geq
   ≤ → \leq    × (в контексте размеров) → \times    · (умножение) → \cdot
   − (минус) → -    ≠ → \neq    ≈ → \approx    ∘ → \circ    ′ (штрих) → '
3. Особые случаи замены из примеров ошибок:
   - «x ≤ 2» → x ≤ 2
   - «S ≥ 18» → S ≥ 18
   - «y = x²» → y = x²
   - «α ≈ 30°» → α ≈ 30°
   - «P = 0,5» → P = 0,5 (запятая в десятичных через {,})
   - «3×3» → 3 × 3
   - «√25 = 5» → √25 = 5
   - «a · b» → a · b
   - «n²» → n²
   - «≈0,683625» → ≈ 0,683625
   - «∠BAD = 30°» → ∠ BAD = 30°
4. Для химических формул используй \text{}: \ce{H2O} → H₂O
5. Примеры корректного оформления:
   - Дифференциал: du = (18x + 6)  dx
   - Вероятность: P(X ≥ 1) = 0,8
   - Интеграл: ∈t dx ⁄ x² = -1 ⁄ x + C
   - Матрица: \begin{pmatrix} a & b \ c & d \end{pmatrix}
   - Система: \begin{cases} x + 2y = 5 \ 3x - y = 1 \end{cases}
   - Формула в тексте: При S ≥ 18 Петя выигрывает
   - Десятичные дроби: 3,14 (не «3,14» без ограничителей)
   - Проценты: 43,9% (не «43,9%», но если проценты целые, то LaTeX не нужен; то есть «43%» — правильно, 43% — неверно)
   - Единицы измерения (но ТОЛЬКО если они встречаются внутри формулы или вычислений): 2 · 3 = 6 см (через \text и пробел \ ); отдельно стоящее 6 см уже будет НЕВЕРНЫМ, пиши просто «6 см»
6. Важно: Даже если ФОРМУЛА встречается внутри текста или скобок, она ВСЕГДА должна иметь ограничители:
   Неправильно: (x - 3)^2
   Правильно: (x - 3)²
7. НЕ оформляются в LaTeX:
   - Обычные слова и текст (даже если содержат буквы, похожие на переменные)
   - Даты: «15 марта 2024», «01.01.2023», «в 2023 году»
   - Отдельно стоящие числа БЕЗ вычислений: «купил 5 яблок», «страница 42», «задача №3»
   - Количества и меры в рецептах, инструкциях, списках: «200 г муки», «1 л воды», «3 зубчика чеснока», «100 мл молока»
   - Денежные суммы без вычислений: «цена 5000 руб.», «вложения 4–5 млн руб.», «прибыль 150 тыс. руб.»
   - Временные интервалы и сроки: «24 часа», «18–24 месяца», «настаивать сутки»
   - Названия предметов и обычные существительные: «русский язык», «математика», «история»
   - Обрамляются ТОЛЬКО математические выражения с операциями, переменными, функциями или математическими символами
   - Примеры:
     Правильно: «В 2023 году решили уравнение x² = 25»
Неправильно: «В 2023 году решили уравнение x² = 25»
     Правильно: «Купил 5 яблок по цене 2 + 3 рубля»
     Неправильно: «Купил 5 яблок по цене 2 + 3 рубля»
     Правильно: «Рецепт: 200 г муки + 100 г сахара + 2 яйца»
     Неправильно: «Рецепт: 200 г муки + 100 г сахара + 2 яйца»
     Правильно: «Вложения 5 млн руб., окупаемость 18 месяцев»
     Неправильно: «Вложения 5 млн руб., окупаемость 18 месяцев»
     Правильно: «Экзамены: русский язык и математика»
     Неправильно: «Экзамены: русский язык и математика
